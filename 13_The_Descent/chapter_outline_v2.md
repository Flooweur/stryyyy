# Chapter 13: Together in the Fall - V2
## (Formerly "The Descent" - Revised Title and Focus)

## Chapter Summary
**REVISED**: After Ed's assassination (Chapter 12), Simera and Liz are both using Mementos heavily. They've established a pattern: using together. Sometimes they experience the same Memento simultaneously, sometimes different ones, but always together—they tell themselves this makes it safer, more controlled. It feels like connection, like shared experience. They start with calmer content but slowly, inevitably, the intensity escalates. The calm ones aren't enough. They're boring. Empty. The intensity calls. They choose more extreme content together. Falling Mementos. Dangerous experiences. Sexual content. The shared experience creates false intimacy—they think they're connecting, but they're actually both numbing, both fragmenting, both descending together. **Liz is escalating faster than Simera did.** Where Simera took months to crave intensity, Liz seems to need more within weeks. Her addiction is faster, deeper, more consuming. Simera notices with growing dread but says nothing—she's using too, they're doing it together, stopping Liz means stopping herself. Time passes. The pattern establishes. They're both lost.
